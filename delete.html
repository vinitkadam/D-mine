Data Deletion

×



Description :

The MySQL DELETE statement is used to delete a single record or multiple records from a table in MySQL.

Syntax :

DELETE FROM table [WHERE conditions];

Example :

1) DELETE FROM contacts WHERE name = 'Johnson';

2) SELECT count(*) FROM contacts WHERE last_name = 'Johnson' AND contact_id < 1000;

Drop Engine

×


Description :

The MySQL DROP TABLE statement allows you to remove or delete a table from the MySQL database.

Syntax :

DROP TABLE table_name;

Example :

Drop One Table
DROP TABLE customers;

Drop Multiple Tables
DROP TABLE customers, suppliers;
